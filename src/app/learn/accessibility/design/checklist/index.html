Checklist of user interface-specific criteria for accessibility

We utilize this checklist to think through and evaluate new patterns and components for SKY UX to ensure they are as accessible as possible. This covers a subset of the WCAG 2.0 success criteria specific to the user interface display and behaviors. It does not cover criteria related to the code or HTML markup used.
